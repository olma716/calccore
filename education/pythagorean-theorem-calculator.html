Калькулятор теореми Піфагора CalcCore — як працює

Цей калькулятор допомагає швидко знайти невідому сторону прямокутного трикутника. Ти можеш обчислити гіпотенузу або один із катетів за відомими значеннями інших сторін.

Формула теореми Піфагора

Основна формула:

  • a² + b² = c²

Де a і b — катети, а c — гіпотенуза.

Як знайти гіпотенузу

Якщо відомі обидва катети, використовують формулу:

  • c = √(a² + b²)

Як знайти катет

Якщо відома гіпотенуза і один катет:

  • a = √(c² − b²)
  • b = √(c² − a²)

Приклад

Для трикутника з катетами 3 і 4:

  • c = √(3² + 4²) = √(9 + 16) = √25 = 5

Що важливо перевіряти

Гіпотенуза завжди повинна бути більшою за будь-який катет. Якщо це не так, введені дані не підходять для прямокутного трикутника.

Часті запитання

Що таке теорема Піфагора?

Це правило для прямокутного трикутника: квадрат гіпотенузи дорівнює сумі квадратів катетів.

Як знайти гіпотенузу?

Треба використати формулу c = √(a² + b²).

Як знайти катет?

Треба використати формулу a = √(c² − b²) або b = √(c² − a²).

Для яких трикутників це працює?

Лише для прямокутних трикутників.

Що буде, якщо гіпотенуза менша за катет?

Такі дані некоректні, тому калькулятор покаже помилку.

Чи підтримуються десяткові числа?

Так, можна вводити і цілі, і десяткові значення.
