Калькулятор квадратного рівняння CalcCore — як працює

Цей калькулятор допомагає розв’язувати квадратні рівняння виду ax² + bx + c = 0. Він автоматично обчислює дискримінант, визначає кількість коренів і показує покрокове рішення.

Формула квадратного рівняння

Загальний вигляд квадратного рівняння: ax² + bx + c = 0, де a ≠ 0.

Дискримінант

Щоб знайти корені, спочатку обчислюють дискримінант:

  • D = b² − 4ac

Від значення дискримінанта залежить кількість дійсних розв’язків.

  • D > 0 — два різні дійсні корені
  • D = 0 — один дійсний корінь
  • D < 0 — дійсних коренів немає

Формула коренів

Якщо D ≥ 0, корені знаходять за формулою:

  • x₁ = (-b + √D) / (2a)
  • x₂ = (-b − √D) / (2a)

Приклад

Для рівняння x² − 3x + 2 = 0 маємо:

  • a = 1, b = -3, c = 2
  • D = (-3)² − 4·1·2 = 9 − 8 = 1
  • x₁ = 2, x₂ = 1

Коли цей калькулятор не підходить

Якщо a = 0, рівняння перестає бути квадратним. У такому випадку треба використовувати калькулятор лінійного рівняння.

Часті запитання

Що таке квадратне рівняння?

Це рівняння виду ax² + bx + c = 0, де a ≠ 0.

Що показує дискримінант?

Дискримінант показує, скільки дійсних коренів має рівняння: два, один або жодного.

Що робити, якщо D менше нуля?

У такому випадку рівняння не має дійсних коренів. Якщо потрібно, можна розглядати комплексні корені.

Чому a не може дорівнювати нулю?

Бо тоді член ax² зникає, і рівняння стає лінійним, а не квадратним.

Чи показує калькулятор покрокове рішення?

Так, він показує підстановку у формулу дискримінанта і формули для коренів.

Чи можна вводити дробові числа?

Так, можна вводити як цілі, так і десяткові коефіцієнти.
